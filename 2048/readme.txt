Ce dossier contient une IA pour le 2048 jouable directement sur un jeu de 2048 existant déjà (sur emulateur de telephone ou un programme déjà fait. 

Il y a donc un lecteur graphique qui lit les données, ensuite l'IA (de type Monte Carlo) calcule le meilleur coup à jouer et enfin le coup est joué directement en faisant bouger la souris.

Cette IA s'inscrit dans ce projet : https://tech.io/playgrounds/17176/recueil-dexercices-pour-apprendre-python-au-lycee/ia-et-jeux-sur-mobiles
